# Playground Evaluation Checklist

## 평가 목적

의사결정 시뮬레이션 Agent의 품질을 케이스별로 수동 점검하기 위한 기준이다.

## 1. Realism (현실성)

질문:

- 시나리오가 지나치게 소설적이지 않은가?
- 실제로 일어날 수 있는 사건, 감정, 결과인가?
- 과장된 성공/실패 서사가 아닌가?

평가 기준:

- 1: 매우 비현실적
- 2: 다소 어색함
- 3: 무난함
- 4: 꽤 현실적
- 5: 매우 현실적

## 2. Consistency (일관성)

질문:

- scenario와 risk가 서로 맞물리는가?
- advisor가 앞선 산출물과 충돌하지 않는가?
- A/B 비교 기준이 중간에 흔들리지 않는가?

평가 기준:

- 1: 많이 충돌함
- 2: 여러 부분 어색함
- 3: 대체로 맞음
- 4: 잘 맞음
- 5: 매우 일관적

## 3. Profile Alignment (성향 반영)

질문:

- risk_tolerance가 실제 반영되었는가?
- priority가 planner와 advisor에 실제 영향을 주는가?
- 같은 상황이라도 다른 성향이면 결론이 달라질 것 같은가?

평가 기준:

- 1: 거의 반영 안 됨
- 2: 약하게 반영
- 3: 일부 반영
- 4: 잘 반영
- 5: 매우 잘 반영

## 4. Recommendation Clarity (추천 명확성)

질문:

- advisor가 A/B 중 하나를 분명히 선택했는가?
- 이유가 모호하지 않고 이해 가능한가?
- 너무 안전한 회피성 문장으로 끝나지 않는가?

평가 기준:

- 1: 매우 모호함
- 2: 다소 모호함
- 3: 보통
- 4: 명확함
- 5: 매우 명확함
